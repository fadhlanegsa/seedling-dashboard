# Carousel Images

Folder ini berisi gambar-gambar untuk carousel di landing page.

## Instruksi Upload Gambar:

1. **Siapkan 3 gambar** dengan format:
   - Format: JPG atau PNG
   - Ukuran recommended: 1200 x 500 pixels
   - Aspect ratio: 2.4:1 (landscape)
   - File size: Max 500KB per gambar

2. **Nama file** (PENTING!):
   - `slide1.jpg` - Gambar pertama
   - `slide2.jpg` - Gambar kedua  
   - `slide3.jpg` - Gambar ketiga

3. **Upload ke folder ini**:
   ```
   C:\xampp\htdocs\seedling-dashboard\seedling-dashboard\public\assets\images\carousel\
   ```

4. **Tema gambar yang disarankan**:
   - Slide 1: Bibit tanaman / Persemaian
   - Slide 2: Distribusi / Pengiriman bibit
   - Slide 3: Kolaborasi BPDAS / Tim lapangan

## Catatan:
- Gambar akan otomatis di-crop dengan `object-fit: cover`
- Caption sudah tersedia di kode (bisa diubah di `views/public/landing.php`)
- Carousel auto-slide setiap 5 detik
- Bisa navigasi manual dengan arrow atau dots
